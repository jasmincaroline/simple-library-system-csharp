# Library Management System (Console App)

This project is a **simple library management system** developed as a **C# console application**.

It was created as part of **Class 11 of the Code RDIversity program**, an educational initiative provided by **RDI Software Brazil** in partnership with **Prosper Digital Skills**.


---

## Features

- Register books in the library
- Register people (users)
- List available books
- Borrow books
- Return borrowed books
- Track book status (Available / Borrowed)
